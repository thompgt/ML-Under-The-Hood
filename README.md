# ML-Under-The-Hood 🧠💻

Welcome to **ML-Under-The-Hood**, an educational repository dedicated to demystifying machine learning algorithms by building them from scratch. No `scikit-learn`, no `tensorflow`, no `pytorch` for the core logic—just pure Python and `numpy`.

## 🎯 Goal
The goal of this project is to provide clear, pedagogical implementations of fundamental ML algorithms. Each implementation focuses on:
1. **Mathematical Foundations**: Understanding the 'why' and 'how' behind the code.
2. **From-Scratch Implementation**: Building the algorithm as a reusable Python class.
3. **Visualization**: Seeing the algorithm in action on toy datasets.

## 🚀 Algorithms Covered
This repository provides step-by-step implementations for the following:
- **Linear Regression**: Implementation via both Gradient Descent and the Normal Equation.
- **Logistic Regression**: Binary classification using the sigmoid function and cross-entropy loss.
- **K-Nearest Neighbors (KNN)**: A simple, instance-based learning algorithm.
- **Decision Trees**: A basic CART implementation using Gini impurity or Information Gain.
- **K-Means Clustering**: Unsupervised learning for grouping similar data points.
- **Multi-Layer Perceptron (MLP)**: A basic neural network with one hidden layer and backpropagation.
- **Naive Bayes**: Gaussian Naive Bayes classification using probability density functions.
- **Support Vector Machine (SVM)**: Linear SVM using Hinge Loss and Gradient Descent.
- **Principal Component Analysis (PCA)**: Dimensionality reduction using eigenvalues and eigenvectors.
- **Random Forest**: Ensemble learning via bagging and random feature selection.
- **AdaBoost**: Ensemble boosting algorithm updating sample weights.
- **Ridge Regression**: Linear regression with L2 regularization.
- **Lasso Regression**: Linear regression with L1 regularization using Coordinate Descent.
- **Gaussian Mixture Models (GMM)**: Soft clustering using Expectation-Maximization.
- **DBSCAN**: Density-based spatial clustering.
- **Agglomerative Hierarchical Clustering**: Bottom-up clustering approach.

## 🛠️ Installation

To get started, clone this repository and install the dependencies:

```bash
git clone https://github.com/[your-username]/ML-Under-The-Hood.git
cd ML-Under-The-Hood
pip install -r requirements.txt
```

## 📚 How to Use
Each algorithm is located in the `notebooks/` directory. Open them using Jupyter Notebook:

```bash
jupyter notebook
```

Explore the notebooks to read the math explanations and step through the code!
